Feasibility of Learning

发表于 2018-11-11 | 分类于 机器学习

我们知道，机器学习是假设存在一个真实的模型，所有的训练数据是从这个模型中产生的，我们希望通过机器学习的方式，学习得到一个模型，希望学习到的这个模型和真实模型之间越接近越好

而机器学习的过程，可以理解为，存在一个假设空间hypotheisis set $H$，通过一个合适的算法algrihtm $A$，从假设空间中选择得到一个合适的$h$，得到模型model $g$，希望这个模型$g$，越接近真实模型$f$越好

在机器学习的训练过程中，我们需要通过建立验证集的方式判断机器学习的模型是否真正的学习到内容；对于分类问题，如果模型在验证集合上的表现／错误率低于50%，我们认为模型是学习到内容的

No Free Lunch

假设训练数据有三个维度，每个维度的取值为$\lbrace0,1 \rbrace$，每个训练样本的标记为$\lbrace o,x\rbrace$

我们可以枚举出假设空间$H$中的每种可能的假设$h$，并且找出在这些$h$中，与训练数据匹配的$h$作为最终训练模型$g$的备选

💡共有3个特征，每个特征有两种取值，即 共有$2^3 =8$种可能的数据；每个数据又有两种可能的标记，即总共有$2^8 = 256$种假设

根据上面的过程，我们可以筛选出8个在与训练数据标记一致的假设；针对这8个假设，我们只能知道，他们在inside training data上是表现好的，但是我们无法衡量在outside training data的部分，这8个模型（假设）是否正确

所以，No Free Lunch的意思即，在没有任何假设前提的情况下，我们其实无法衡量一个模型是不是好的模型，但是这并不代表机器学习模型是不能学习的，只不过需要存在一定的假设

Inferring Something Unkown

假设我们现在有一个罐子，罐子里有橙色和绿色的小球（无数个），我们的目标是找到橙色小球占总小球树数目的比例。

假设罐子里橙色小球的比例为$\mu$，对应绿色小球的比例为$1-\mu$

💡如果罐子里的小球数目是无穷多的，则在抽样过程中，直接抽样即可；如果小球数目是有限的，则在抽样过程中采用有放回抽样，保证罐子里的橙色球数目一直为$\mu$

我们从罐子里抽取N个样本，N个样本里，橙色小球的比例为$\nu$，对应的绿色小球的比例为$1-\nu$

我们称$\nu$为in-sampl比例，$\mu$为out-of-sample的比例

in-sample的$\nu$无法告诉我们out-of-sample的$\mu$的准确值，但是我们可以得知，当抽样的样本数目N非常大时，in-sample的$\nu$和out-of-sample的$\mu$是非常接近的（大数定理）

💡这有点类似人口普查，在调查某个省份的男女比例，我们不必统计全省的男女各有多少人，只需要采用合理的抽样方式，抽出N个样本，统计N个样本中男女比例多大即可 （科学的抽样方式，个人理解是，不能在女子学校里抽N个人，这样N个样本都是女生）

Hoeffding’s Inequality

hoeffding不等式证明了上述观点。

假设抽样样本为N，N中橙色小球的比例为$\nu$；整体样本中橙色小球的比例为$\mu$，则

$P[|\nu-\mu|>\epsilon] \leq 2exp(-2\epsilon^2N)$

即，$\nu$和$\mu$之间的相差的$\epsilon$ 的概率小于$2exp(-2\epsilon^2N)$

当N越大时，这个概率值越小

这表明$\nu = \mu$是在PAC假设下成立的 （PAC=probably approximately correct）有很大概率是正确的

（1）这个不等式与$\mu$无关，在不知道$\mu$的情况下，我们可以估算出$\nu$与真实值之间的差距的一个上界；

$\nu$受到$\mu$的影响，$P[|\nu-\mu|>\epsilon]$的概率是受到$\mu$的影响，但是这个概率的上界与$\mu$无关

（2）这个不等式对所有的N和$\epsilon$都适用

（3）N越大，$\epsilon$越松弛（越小）表明$\mu$与$\nu$有更大的概率越接近

Verification Process

💡现在我们讨论在假设空间只有一个假设时，hoeffding不等式与机器学习之间的关系

💡假设空间只有一个假设 <=> 我们从假设空间中选择其中一个假设作为我们最终的模型

类比上面小球的例子，我们现在有一个真实的函数（数据概率分布），不断的产生数据（bin里的小球）

我们从bin里抽样出小球的过程 <=> 我们从真实函数产生的多个样本中抽出多个训练样本

我们利用假设空间中的这个假设h，作用在训练样本上；如果$h(x) = f(x)$，则表明模型分类正确（以分类问题为例），我们将小球涂绿色；如果$h(x) \neq f(x)$ ，则表明分类不正确，我们将小球涂成橙色

上面小球的例子是通过样本中橙色小球的比例估测真实bin里橙色小球所占的比例；

同样，我们在h分类过程中，通过计算分类错误率（橙色小球比例），可以推测出在真实的所有数据中，h分类错误的概率（橙色小球占真实世界所有小球的比例）

数学表达式为：

$E_{in}(h) = \frac{1}{N}\sum_{n=1}^N[h(x_n) \neq y_n]$ h在训练样本上分类错误率，相当于$\nu$

$E_{out}(h) = \varepsilon_{x\sim p}[h(x_n)\neq f(x)]$ h在真实样本上分类错误的概率

💡在$E_{in}$中，样本的真实标记为y；但是在$E_{out}$中，样本的真实标记为f(x)；因为训练数据中可能存在噪音

因此，对于每个固定的h，当训练样本很大时，in-sample error $E_{in}$可以近似约等于out-of-sample error $E_{out}$ ，即，$E_{in}$和$E_{out}$在PAC假设下是近似相等的：

$P[|E_{in}(h)-E_{out}(h)|>\epsilon] \leq 2exp(-2\epsilon^2N)$

这个过程是验证过程而不是学习过程，因为现在只有一个固定的假设h，我们讨论其$E_{out}$多大；但是在实际的模型训练过程中，我们应该通过一定的演算法从假设空间选择一个合适的假设h，而不是固定选择一个假设h

=> 当这个假设h在训练样本上表现并不是很好时，我们依然选择这个假设h作为我们的模型，那么我们的学习过程是失败的

Learning Process

真正的学习过程需要在假设空间$H$中选择一个最合适的假设。

一个罐子bin代表整个输入空间$X$，不同的假设对应了罐子里小球的不同涂色（小球的涂色与假设密切相关）

Hoeffding不等式只是表明，在固定一个假设的情况下，这个假设在in-sample error $E_{in}$ 和 out-of-sample error $E_{out}$上是PAC的

但是并不能保证$E_{in}$是小的

因此，整个的学习过程是在一个假设空间$H$中找到一个最合适的假设；而当出现多个假设，需要从中选择一个假设时，出现红色小球的概率会变高，原有的bound $2exp(-2\epsilon^2N)$会失效

以抛硬币为例，我们现在抛150个硬币 （每个硬币代表假设空间中的一个假设），每个硬币抛5次（当作用训练样本进行模型训练），假设硬币正面表明训练正确，硬币反面视为训练错误。如果一个硬币的连续5次均出现正面，这个硬币是否是最好的硬币？（最优的假设）

不是！抛一个硬币连续出现5次的概率$p(连续出现5次) = (\frac{1}{2})^5 = \frac{1}{32}$

则$p(至少有一次不是正面) = 1-\frac{1}{32} = \frac{31}{32}$

则至少有一个硬币连续出现5次正面的概率$p = 1-(\frac{31}{32})^{150}> 99\%$

从概率的角度说明，出现连续五次硬币正面的概率很高，这并不能说明这个硬币／模型就是好的

即，当有选择出现时，会出现偏见，其实这150个硬币是一样的，但是我们会选择出现5次正面的那个硬币，即会选择看起来很好的模型／假设； hoeffding不等式表明，在只有一个假设时，in-sample error和out-of-sample error相差很大的概率很低

=> 如果只有一个硬币，连续出现五次正面（in-sample error为0）的概率是$\frac{1}{32}$，但是出现150个硬币可以选择时，连续出现五次正面的概率大于99%

因此，我们需要讨论，我们现在已经从假设空间中选择出一个最合适的假设g，其in-sample error和out-of-sample error之间的误差上界：

$P[|E_{in}(g)-E_{out}(g)|>\epsilon] \leq$

$P[|E_{in}(h1)-E_{out}(h1)|>\epsilon] \; or \; P[|E_{in}(h2)-E_{out}(h2)|>\epsilon] \; or \; P[|E_{in}(h3)-E_{out}(h3)|>\epsilon] \; or \ $

$……P[|E_{in}(h_M)-E_{out}(h_M)|>\epsilon] ​$

$\leq \sum_{m=1}^M P[|E_{in}(h_m)-E_{out}(h_m)|>\epsilon]​$

假设空间中存在一个最好的假设g，这个假设满足$P[|E_{in}(g)-E_{out}(g)|>\epsilon]​$的概率，小于等于整个假设空间中每个假设发生这个$P[|E_{in}(h)-E_{out}(h)|>\epsilon]​$事件的概率

$P[|E_{in}(g)-E_{out}(g)|>\epsilon] \leq 2Mexp(-2\epsilon^2N)$

💡此处我们讨论假设空间中的假设个数是有限个M

💡如果我们以假设空间假设的个数M作为衡量指标，来判断模型的复杂程度，则模型越复杂，对应的假设空间越大，即M越大；模型越复杂，代表模型越能更好的拟合in-sample error，但是，对应的in-sample error和out-sample error出现差距很大的概率的上限也会变大

上述upper bond表明，当训练样本数目N足够多时，训练样本对假设空间中的每个假设都是安全的，所以我们可以选择in-sample error最小的那个假设

总结一下，hoeffding不等式控制的是泛化误差，我们可以选择假设空间中在in-sample上表现最好的一个假设，但是我们如何保证在 out-of sample的地方，这个假设依然是有效的，即，我们采用hoeffding不等式证明，in-sample error和out-of-sample error之间相差$\epsilon$的概率会被bound住
