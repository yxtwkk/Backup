Convolutional Neural Network

发表于 2018-07-08 | 分类于 机器学习

卷机神经网络 CNN（Convolutional Neural Network）在图像识别和特征提取方面有很大的优势，也正是因为其强大的网络结构，也相继出现如text-cnn之类的类cnn网络用于文本特征提取。

CNN也代表着我们可以根据自己的需求，设计不同的神经网络

（我们可以通过设定不同的参数值来使得全连接神经网络做各种不同的变形，类似于，全连接神经网络是一个大的函数空间Function Set，而不同的神经网络模型／深度学习模型都是函数空间中的一个函数，而这个函数又自身可以代表一个Function Set，通过设定不同的网络权值，得到不同的函数）

背景Background

（1）我们其实不需要观察整个图像，就可以识别其中的一些特征；比如，我们可以只看一部分，就可以判断出图像是否包含鸟喙，而不需要观察整张图像

（2）在一幅图像中，同样的pattern可能出现在不同的位置上，所以并不需要在每个位置上都设置参数/权重，在相同pattern的地方可以使用同样的参数

（3）下采样Subsampling不会改变image的形状

通过下采样的方式使得图片的pixel（像素点）更小，这样我们需要用到的参数也就更少

CNN网络的网络结构就是基于以上三点设计的

CNN网络结构

CNN主要分为卷积层（Convolution），池化层（Max Pooling），Flatten层，全连接层；其中卷积层和池化层可以重复多次

卷积层（Convolution）：我们不需要观察整个图像，就可以发现其中的pattern；一些pattern会在图像的多个位置出现

池化层（MaxPooling）：下采样并不会影响我们对图像的判断

卷积层和池化层主要用来提取特征，全连接层利用提取到的特征进行学习（分类／回归）

卷积层 Convolution

假设有一个6*6的图像（以单通道图像为例），每个位置代表一个像素点pxiel值

卷积层是通过filter在图像上扫过，形成feature map用来提取图像特征；

每个filter是一个矩阵（假设3*3），需要设定步长stride（用来表示一个filter在图像上移动的距离），filter移动过程中要和所覆盖的位置的图像的像素点组成的矩阵做内积（即，对应位置相乘并相加）；

当一个filter对整个图像完成一次扫描以后，就会得到一个feature map；多个filter就会得到多个feature map

如图所示，假设我们利用这样一个filter提取image特征，这个filter所提取的特征就是斜对角线为1的pattern（斜对角线为1，输出的feature更大）

完成一次filter以后，可以发现，利用相同的filter，可以在图像的不同位置发现一样的pattern，比如，在左上角和左下角的pattern是一样的

💡filter层也可以称之为kernel层，其实就是我们需要学习的参数

💡感受野（Receptive Field）：在卷积神经网络CNN中，决定某一层输出结果中一个元素所对应的输入层的区域大小，被称作感受野receptive field

💡对于多通道图像而言，比如RGB图像，每个pixel位置点有三个值来表示（R/G/B，分别代表RGB的强度，每个值为一个channel），则对应的filter也需要有三个通道来表示

💡对于多通道图像来说，每个filter会对应的变成一个方体结构，在对图像进行特征提取时，也是进行内积运算，即对应位置相乘之后再相加

💡一般而言，卷积网络中，靠近输入图像的卷积层filter个数较少，靠近输出层／flatten层的卷积层filter个数应该较多。因为一般靠近输入图像提取的是一些简单的图像pattern特征，所以不需要更多的filter个数

卷积层 vs 全连接层

一个卷积层的filter的每一个位置上对应的就是一个参数，如果是3*3的filter，其实就是9个参数，比照全连接网络，以上面的filter 1为例

（1）对于下一层的输入，其只连接上一层的几个位置（像素点），而不是连接多个像素点

（2）图示中相同颜色的边代表的是相同的权重（即filter 1对应的一个位置），利用相同的权重参数，但是改变不同的输入（不同位置的像素点），得到不同的输出

CNN通过这种方式得到更少的参数！！

Pooling

池化就是选择一些元素代表其中一个元素，可以用Max Pooling或者Average Pooling

以上面的网络为例，filter 1经过图像以后形成的feature map，每4个pixel可以选择其中最大的作为代表，形成一个更小的图片

💡卷积层的每个filter会形成一个channel

💡原始图像经过一 次卷积一次池化以后，会得到更小的图像

Flatten

flatten层是将经过卷积和池化之后的图像拉直成一个向量，输入全连接层进行分类／回归学习

What does machine learn

我们如何判断机器学习学习到的内容？

对于分类或者回归问题，我们可以建立一个baseline，随机对训练样本进行赋值／标记类别，如果得到的结果比用训练好的模型赋值／标记类别的结果更好，则表明模型没有学习到什么东西

对于神经网络问题而言，我们可以采用如下的方法：

（1）对于第一层卷积层，我们可以通过将其卷积后得到的结果输出（即特征提取结果），这样就可以看出第一层卷积后学习到的特征

💡我们可以通过这种方法，利用学习好的模型，进行迁移学习和特征提取

（2）对于高层的卷积层higher layer，将其卷积层输出是无法人为观察得到一些可以理解的特征，但是我们可以找到什么样的输入会使得卷积层的更被激活

“更被激活”这个判定条件可以这样表示：

我们假定，图像经过经过第k个filter之后，得到的是11*11的矩阵，则第k个卷积层被激活的程度可以这样表示：

Ref：Ross Girshick, Jeff Donahue, Trevor Darrell, Jitendra Malik, “Rich feature hierarchies for accurate object detection and semantic segmentation”, CVPR, 2014

$a^k = \sum_{i=1}^{11}\sum_{j=1}^{11}a^k_{ij}$

$a^k$表征第k个filter被激活的程度，则对于第k个filter，我们可以找到一个image x使得$a^k$最大

$x^* = arg\underset{x}{max}\;a^k$

可以通过固定参数，通过改变x，找到使得$a^k$最大的图像

以下是每个filter找到的使其被最激活的输入的图像，比如，对于第一排第三个filter，当输入图像全部是斜条纹时，其最被激活

（3）我们可以用同样的方法找到全连接层的神经元学习到的内容

对于全连接层而言，虽然其输入的维度要远小于原始图像，但是其本质上看到的是整个图像

我们可以找到一个输入的原生图像，使得全连接第j个神经元$a_j$的输出可以得到最大值

（4）假设全连接层的最后输出是图像的类别，是基于softmax的一个概率向量

分类任务是图像数字识别，通过上述方式找到的可以使$y_i$最大的输入图像如下图所示：

💡由此可见，深度学习网络是很容易被“愚弄的”

Ref：https://www.youtube.com/watch?v=M2IebCN9Ht4

当然，这种被“愚弄”的地方有一个好处就是，深度学习可以找到一些我们无法观察到的特征，可以学习到我们无法学习到的可能有用的信息

我们可以采用正则化的方法解决这个问题：

通过这种方法找到的x，$x^* = arg\underset{x}{max}\;y^i$，可以通过加入正则项的方法优化：

$x^* = arg\underset{x}{max}\;（y^i-\sum_{i,j}|x_{ij}|）$

💡对图像x的一些pixel作出一些限制，x的某些pixel虽然可以让$y_i$最大，但其存在使得图像不是数字，数字只有在某些图像区域有笔画

💡加入的优化是1范式正则化，即找出的图像绝大部分是没有笔画的

💡也可以加入其他的一些约束，比如相邻的pixel应该是同样的颜色等
